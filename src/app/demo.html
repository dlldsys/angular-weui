Angular WeUI 是一个基于 Angular 框架的 UI 组件库，旨在为开发者提供与微信原生视觉体验一致的 UI 组件。

组件库包含了按钮、表单、对话框、列表、网格等常用组件，并且支持主题定制和暗黑模式。

我们希望这个组件库能够帮助开发者快速构建高质量的移动端应用。
